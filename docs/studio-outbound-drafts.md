## Reply triage when responses come in

| Reply pattern | Response |
|---|---|
| "Yes, send a free sample" | Brief them properly. Confirm product + 1 hook angle. Run the pipeline. Send within 48h. This IS the lead. |
| "How much exactly?" | Quote the from-rate from the page. Offer a fixed-price for one specific brief (don't open custom). |
| "Looks AI" / "We need real creators" | Don't argue. "Fair — keeping you in mind if that ever changes" + close. |
| Silence (no reply in 7 days) | One follow-up: "Brief reminder if useful — [link]". Then drop. |

## After the first 5

If response rate ≥ 1 in 5 → repeat with 5 more, same archetypes, refine subject lines based on opens.
If response rate is 0 → don't send 5 more. Something's off. Diagnose the page (does the OG card render? is studio.html actually loading? is the price scaring people off?) before more sends.
