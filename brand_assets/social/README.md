# AireyAI Social Headers

- `twitter-header-1500x500.png` is the upload-ready X/Twitter header, with `Airey` in the site heading color and `AI` in the site green.
- `twitter-header-sitecase-1500x500.png` preserves the exact live-site casing: `AireyAi`.
- `twitter-header-source.html` is the editable source. It uses the same `Hanken Grotesk`, `#061b31` text, and `#0b8a5b` green as the live AireyAI website brand mark.
- Run `node render-twitter-header.mjs` from this folder after editing the source.
