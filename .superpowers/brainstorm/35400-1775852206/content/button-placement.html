Where should we add buttons?

The AireyAi site has 5 sections. Click all the areas where you'd like to add or improve buttons.

A

Hero Section

Top of the page — add social proof links, a secondary action, or a "Watch demo" button

B

How It Works

"Three steps" section — add a CTA button after the steps so visitors can take action immediately

C

What's Included

Features section — add buttons to each feature card, or a "Get started" below the list

D

Navigation Bar

Add nav links (Services, Portfolio, Pricing) with an improved CTA button layout

E

Floating / Sticky Button

A persistent "Book a call" or chat button that follows the user as they scroll
